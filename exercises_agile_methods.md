# The Board

* 2D playing field of *cells*
* Each cell is either dead or alive
* Neighbours preserve, kill or spawn alive cells


	###		o cell
	#o#		# neighbour
	###

--

# The Cells

* 2 Neighbours: Survive
* 3 Neighbours: Spawn or Survive
* `or die`
